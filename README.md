# Портфолио системного аналитика

В данном репозитории собраны учебные и практические работы, выполненные в рамках обучения по направлению «Системный анализ».

## Содержание

### 01. Роль системного аналитика
- Конспекты и практические задания по роли системного аналитика
- Анализ задач и обязанностей аналитика

### 02. Требования, Use Case, User Story
- Сбор и анализ требований
- Use Case диаграммы
- User Story
- Спецификация требований

### 03. Диаграммы последовательности и деятельности
- Sequence Diagram
- Activity Diagram
- Моделирование бизнес-процессов

### 04. UML-диаграммы
- Диаграммы классов
- Диаграммы состояний
- Другие UML-диаграммы

## Используемые инструменты

- Draw.io
- PlantUML
- Figma
- GitHub

## Навыки

- Сбор и анализ требований
- UML
